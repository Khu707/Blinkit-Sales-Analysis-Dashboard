# Blinkit-Sales-Analysis-Dashboard
This project showcases an interactive Power BI dashboard built using the BlinkIT Grocery Sales dataset. The objective is to analyze sales performance, outlet characteristics, product categories, and customer ratings to derive meaningful business insights.

The raw data was cleaned and prepared in **Microsoft Excel**, and the final dashboard was developed in **Power BI** using DAX measures and interactive visualizations.

### 🔍 Key Insights

* Total Sales: **$1.20M**
* Number of Items Sold: **8,523**
* Average Sales per Item: **141**
* Average Customer Rating: **3.9**
* Analysis of Outlet Size, Location, and Type
* Product Category Performance
* Fat Content Distribution
* Outlet Establishment Trends

### 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel
* Data Cleaning & Transformation
* DAX (Data Analysis Expressions)
* Business Intelligence & Data Visualization
